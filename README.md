# ANTLR4 parser for Python 2.7.18 &nbsp; [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

### About files:
 - PythonParser.g4
   is the ANTLR4 parser grammar that based on the last official [Python 2 grammar](https://docs.python.org/3.8/reference/grammar.html) which not yet written in [PEG](https://docs.python.org/2.7/reference/grammar.html).

 - PythonLexerBase
   handles the Python indentations.
   
 - Example files: [Python 2.7 Standard Lib](https://github.com/certik/python-2.7/tree/master/Lib)
